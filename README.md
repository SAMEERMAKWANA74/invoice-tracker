# Invoice Tracker

A simple **Invoice Tracker** web application built with **React**. This app allows users to create, review, and download invoices quickly and easily.

---

## **Features**

- Add company (Bill from) and client (Bill to) details.
- Add multiple invoice items with quantity and price.
- Calculate **subtotal, discount, tax, and total** automatically.
- Select currency for the invoice.
- Download invoice as a PDF (future implementation possible).
- Clean and responsive UI.
